# CRUD-Empleados
Aplicación movil creada en android con lenguaje kotlin.
La aplicación es un CRUD en la cual el usuario puede ingresar registros de los empleados, modificarlos, leerlos y eliminarlos.

La aplicación inicia y se observan 2 campos requeridos para poder agregar al empleado, si se llenan los campos y se pulsa en agregar
empleado, la aplicación guardará el registro de ese empleado.

![CRUD1](https://user-images.githubusercontent.com/108247794/228479914-0186bcb0-4d7c-43b6-9a80-9d73f780d069.png)

![CRUD7](https://user-images.githubusercontent.com/108247794/228480035-d965f3ec-5e68-4957-9f83-dcfcd33b9308.png)

Si al se abre la aplicación y se pulsa en el botn de ver empleados, la aplicación mostrará todos los empleados registrados.

![CRUD2](https://user-images.githubusercontent.com/108247794/228480333-051bce45-82b3-423c-869a-20807804c357.png)

Si se pulsa en la imagen del lápiz que tiene el registro del empleado a la derecha, el usuario puede modificar a dicho empleado,
se tiene que modificar el/los campo/os que se modificarán.

![CRUD3](https://user-images.githubusercontent.com/108247794/228480859-8a289cec-ff55-4d47-b241-61703a2afbb2.png)

![CRUD4](https://user-images.githubusercontent.com/108247794/228480892-553546b4-61e1-4dea-823e-a87613c1e2e1.png)

Si se pulsa en la imagen del basurero que tiene el registro del empleado a la derecha, el usuario podrá elimiar a dicho empleado,
pero antes de eliminarlo, aparecerá una alerta donde indicará si realmente se eliminará dicho empleado o no, si se pulsa que si,
dicho empleado se elimina por completo, si se pulsa que no, el registro del empleado se mantiene.
